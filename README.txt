tohu-generator is an add on to tohu (https://www.jboss.org/tohu).
It provides the ability to use spreadsheets to define a web based 
questionnaire using Drools tempates.

To install just run mvn install. Then download generator-example 
and build that. Go into generator-example-web and run mvn jetty:run.

Then in your browser go to http://localhost:8080/generator-example-web
